# StreetBuilder
Um jogo simples de construção de ruas com simulação de trânsito usando mapas reais (OpenStreetMap).

## Como jogar
1. Abra o arquivo `index.html` no navegador.
2. Clique no mapa para adicionar pontos e criar ruas.
3. Carros vermelhos aparecerão e circularão pelas ruas criadas.

## Tecnologias
- JavaScript
- Leaflet.js
- OpenStreetMap
